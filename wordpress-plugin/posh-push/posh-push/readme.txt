Send web push notifications to your WordPress site visitors. Easy setup, auto-notifications on new posts, and smart targeting.

== Description ==

Posh Push Notifications connects your WordPress site to the Posh Notification System — a powerful push notification platform that helps you engage visitors with targeted web push notifications.

**Features:**

* One-click setup with API key
* Automatic push notifications on new post publish
* Customizable subscription widget
* GDPR-compliant consent collection
* Smart subscriber segmentation
* Analytics and click tracking
* A/B testing support
* Minimal database usage in WordPress

== Installation ==

1. Upload the plugin to `/wp-content/plugins/posh-push/`
2. Activate through the 'Plugins' menu
3. Go to Posh Push > Settings
4. Enter your API key from the Posh Push dashboard
5. Done! The subscription widget will appear on your site.
